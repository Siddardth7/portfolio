## Visual Direction

### Aesthetic
**Dark, precise, technical.** Think: aerospace instrument panel meets modern developer portfolio. Not corporate, not playful — confident and rigorous.

Keywords: dark mode · clean · structured · professional · technical credibility

### Current Color System (existing site — can evolve but keep dark base)
| Token | Hex | Role |
|---|---|---|
| Background primary | `#0a0a0b` | Page background |
| Background secondary | `#111113` | Section alternates |
| Card background | `#161618` | Cards |
| Accent primary | `#3b82f6` | Blue — primary CTA, highlights |
| Accent secondary | `#60a5fa` | Lighter blue |
| Accent warm | `#f59e0b` | Amber — "Built in Code" section accent |
| Accent success | `#10b981` | Green — credentials, positive metrics |
| Text primary | `#fafafa` | Main text |
| Text secondary | `#a1a1aa` | Supporting text |
| Text muted | `#71717a` | Labels, dates |

You may propose a refined or evolved palette — but the dark base (`#0a0a0b` range) and blue primary accent are intentional and should be preserved.

### Typography (existing — can refine)
- **Headings:** Space Grotesk (700 weight) — sharp, geometric, technical
- **Body:** Inter — clean, readable
- **Monospace/labels:** JetBrains Mono — for badges, dates, stat values, code-adjacent content

### Key Design Moments to Nail
1. **Hero** — the headline "Engineering Quality Into Every Process" should feel bold and confident. The bio is dense — needs good line height and breathing room.
2. **Built in the Lab vs Built in Code** — these two sections should be visually connected but distinguishable. Suggestion: Lab = green accent treatment, Code = amber accent treatment.
3. **Six Sigma Green Belt** — must be prominent in the Skills section. This is a key credential — treat it like a badge or certificate, not just a text chip.
4. **Stats bar** — 3 stats only: 7 Projects · 3 Internships · Six Sigma GB. Clean, large numbers, not cluttered.
5. **"Built in Code" cards** — each must show a "Live Demo" button. These are deployed apps — that link is a major credibility signal.

### Navigation
- Fixed/sticky navbar
- Logo: "SP" (existing moon-styled circle — keep or refine)
- Links: Built in the Lab · Built in Code · Experience · Skills · Contact
- Resume button (primary style) on right side of nav

### What to Remove from Current Design
- Spacecraft emoji animations in the hero (too playful, hurts professionalism)
- Philosophy section (Precision Under Pressure / Theory Meets Reality / Constraints Spark Innovation) — removed entirely
- The broken contact form alert() — replace with proper form design
- "© 2025" — update to 2026

### Responsive Requirements
- Desktop (1440px): Full multi-column layouts
- Tablet (768px): Stack to 2 columns or single column where needed
- Mobile (375px): Single column, nav collapses to hamburger
